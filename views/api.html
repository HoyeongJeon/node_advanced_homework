API 페이지입니다.


    GET: http://localhost:3000/api/products (상품 목록 조회)

    POST: http://localhost:3000/api/products (상품 작성)

    body에 {title,content, author, password} 넘겨줘야함

    GET: http://localhost:3000/api/products/:productId (상품 상세 정보 확인)

    URLParameter로 {productId} 넘겨줘야함

    PUT: http://localhost:3000/api/products/:productId/edit (상품 정보 수정)

    URLParameter로 {productId} , body로 {title, content, password, status} 넘겨줘야함

    DELETE: http://localhost:3000/api/products/:productId/delete (상품 삭제)

    URLParameter로 {productId} , body로 {password} 넘겨줘야함
